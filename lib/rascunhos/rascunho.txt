*** GRAFICO DE RADAR ***

- Lista de sets
	- Cada set tem uma lista de valores
	- Cor preenchimento de cada set
	- Cor da borda de cada set
	- Raio dos pontos que ligam cada set
- Titulo (estilo, posição, cor)
	- Distancia dos titulos para a borda do radar
	- Estilo dos titulos (cor e tamanho da fonte)
- Cor de fundo
- Formato (cicular ou polygon)
- Espessura e cor da borda
- Quantidade de "tick", niveis
- Cor e tamanho das descrições dos ticks
- Cor do grid
